Explorer for Angular 2
---------------------------
Shows how to use all the Wijmo 5 controls in Angular 2 application.

The sample uses Angular 2 Beta.1 version.
System requirements:
- To run the sample from Visual Studio you need Visual Studio 2015 with TypeScript version 1.7 or higher.
- Alternatively you may host the sample in a Web Server and run its default.htm page.
- The sample was tested in the following desktop browsers: Chrome, Edge, FireFox, IE11.
