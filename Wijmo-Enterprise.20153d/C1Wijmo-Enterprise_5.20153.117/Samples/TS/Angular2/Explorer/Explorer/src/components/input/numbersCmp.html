InputNumber

The InputNumber control allows users to edit numbers. It prevents users from accidentally entering invalid data and formats the number as it is edited.

You can specify a valid range using the min and max properties, and provide spinner buttons (showSpinner property) that increase or decrease the value with a click. You can also use the format property to specify the .NET-style format string

These samples use the WjInputNumber component and Angular 2.
