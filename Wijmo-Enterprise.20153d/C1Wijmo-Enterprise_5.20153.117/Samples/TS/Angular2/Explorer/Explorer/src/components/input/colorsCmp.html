InputColor and ColorPicker

The InputColor control allows users to enter and edit HTML colors by typing any valid CSS color string into the control or using a drop-down that displays a ColorPicker control.



ColorPicker

The ColorPicker control shows a customizable color palette and editors for the color components (hue, saturation, brightness, and transparency).

The InputColor control allows users to enter and edit colors by typing or by using a ColorPicker in a drop-down.
