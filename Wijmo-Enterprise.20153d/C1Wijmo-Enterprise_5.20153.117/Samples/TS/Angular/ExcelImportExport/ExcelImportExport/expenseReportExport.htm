This example exports a hierarchical array of data, containing employees' expenses, to an Excel xlsx file using the XlsxConverter.exportToFile method.

The array contains items with employee data, wherein each employee item contains a child array with employee's expenses. The sample creates a Workbook Object Model representation of the xlsx file data, provided by the IWorkbook interface. It is then passed to the XlsxConverter.exportToFile method to save it as an Excel xlsx file on the local disk.

The created workbook contains a separate sheet for each employee, with employee's personal data and expenses table.
