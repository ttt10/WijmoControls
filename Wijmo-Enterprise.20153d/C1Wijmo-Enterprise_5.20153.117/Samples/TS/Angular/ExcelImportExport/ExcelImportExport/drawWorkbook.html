This sample implements a simple Excel workbook viewer that loads xlsx file content and displays it on the page.

The sample loads an xlsx file content using the XlsxConverter.import method, which returns a Workbook Object Model representation of the file provided by IWorkbook interface. After that, it walks through the content of the current workbook sheet provided by IWorkSheet interface, and creates an HTML table that visualizes the content.

The sample also creates tabs that allow switching between workbook sheets. It is implemented using the BootstrapJS tab-like styling of the <li> elements with the applied ng-repeat directive that iterates through the IWorkbook.sheets collection.
