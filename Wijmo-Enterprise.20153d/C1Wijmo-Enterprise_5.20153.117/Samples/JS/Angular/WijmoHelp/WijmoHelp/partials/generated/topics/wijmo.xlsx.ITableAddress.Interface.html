ITableAddress Class

Module
wijmo.xlsx
File
wijmo.xlsx.js

Defines a cell index with zero-based row and column components, as well as the properties indicating whether the index component is absolute (for example "$D") or relative (for example "D").

Properties

Name Type Description
F absCol boolean Indicates whether the original column index is absolute (for example "$D") or relative (for example "D").
F absRow boolean Indicates whether the original row index is absolute (for example "$15") or relative (for example "15").
F col number A zero-based column index.
F row number A zero-based row index.
