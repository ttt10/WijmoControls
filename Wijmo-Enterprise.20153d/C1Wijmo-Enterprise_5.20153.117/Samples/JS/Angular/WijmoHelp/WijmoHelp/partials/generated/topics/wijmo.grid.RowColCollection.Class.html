Abstract class that serves as a base for row and column collections.

Constructor

Name Parameters Description
constructor​(grid, defaultSize)

grid (FlexGrid): The FlexGrid that owns the collection.
defaultSize (number): The default size of the elements in the collection.

 Initializes a new instance of a _RowColCollection.

Properties

Name Type Description
defaultSize number Gets or sets the default size of elements in the collection.
frozen number

Gets or sets the number of frozen rows or columns in the collection.

Frozen rows and columns do not scroll, and instead remain at the top or left of the grid, next to the fixed cells. Unlike fixed cells, however, frozen cells may be selected and edited like regular cells.
maxSize number Gets or sets the maximum size of elements in the collection.
minSize number Gets or sets the minimum size of elements in the collection.
