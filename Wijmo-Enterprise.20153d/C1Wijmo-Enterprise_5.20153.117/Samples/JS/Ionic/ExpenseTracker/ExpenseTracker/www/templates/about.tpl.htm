The purpose of the Expense Tracker sample is to demonstate Wijmo 5's AngularJS directives in a Cordova-based application.

The Wijmo 5 controls used in this sample are:

  • Calendar: Used for selecting a date when creating a new expense.
  • ComboBox: Used for selecting a category when creating a new expense.
  • CollectionView: Used with the FlexGrid to perform bulk edits.
  • FlexChart: Used to display a high level overview of where money is being spent.
  • FlexGrid: Used to display, edit, and sort all expenses that have been created.
  • InputNumber: Used to edit numeric values when editing the budget and creating an expense.

The Expense Tracker sample is a hybrid mobile application and was created using the Ionic Framework, AngularJS, Apache Cordova, HTML5, FontAwesome, and, of course, Wijmo 5. Because the Expense Tracker is a hybrid mobile application, it is best viewed on a mobile device as an app; however, any WebKit-based browser should be fine as a substitute.
